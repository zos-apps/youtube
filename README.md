## About

Video streaming platform with millions of videos.

## Features

- Video streaming
- Subscriptions
- Playlists
- Comments
- Live streams

## Installation

### From App Store
Open the **App Store** in zOS and search for "YouTube".

### Via npm
```bash
npm install @zos-apps/youtube
```

## Usage

- Search for videos
- Subscribe to channels
- Create playlists

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Space` | Play/Pause |
| `F` | Fullscreen |
| `M` | Mute |
| `J/L` | Skip |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
